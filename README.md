# Mechanical Gripper Arm

## 📋 Project Overview
Precision mechanical gripper arm with multi-axis articulation for robotic applications. This project includes detailed CAD models, assembly drawings, and technical specifications.

## 🎯 Features
- Multi-axis articulation for precise control
- High grip force with minimal power consumption
- Modular design for easy customization
- Compatible with standard robotic platforms

## 📐 Specifications
- **Material**: Aluminum alloy with steel joints
- **Weight**: ~2.5 kg
- **Grip Force**: 50-100 N
- **Operating Temperature**: -10°C to 60°C

## 📁 Files Included
- `gripper-arm-assembly.step` - Complete assembly model
- `gripper-arm-drawing.pdf` - Technical drawings
- `specifications.txt` - Detailed specifications
- `images/` - Design screenshots and renders
